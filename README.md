# CNCGToolKit
这个开源项目是由中国CGTD在线编程实例总结出来的。允许任何人下载、使用、修改、再发布。

项目提交规则：
- 本项目仅接受一下三种commit描述：
    + 漏洞修补，格式为"Debug:<Your Message>" 
    + 新内容，格式为"NewFeature:<Your Message>"
    + 内容调整，格式为"Fix:<Your Message>"
- 本项目内python模块应当尽量符合PEP8规范。
